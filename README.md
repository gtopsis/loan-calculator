# loan-calc

## Description

A simple loan calculator based on final price of a car, desired down payment and the duration of loan repayment.

## Frameworks used

- Vue.js v2.6.11 (with vue-router v3.2)
- Bootstrap 5.0.2
## Project setup

```
npm install
```
